# Getting Started

## Usage

There is a hard rate limit of 60 requests per minute for both authenticated and non-authenticated requests.

<Steps>
### Generate an API Key

You can obtain an API key through the [accounts portal](https://titan.tf/settings/api).

### Authentication

All API endpoints are authenticated using Bearer tokens.

Use your API key to generate an access token. Access tokens are valid for 1 hour and can be used to make requests to the API.

### Refresh Tokens

Refresh tokens are valid for 30 days and can be used to generate a new access token.

</Steps>
